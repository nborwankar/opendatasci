Welcome to OpenDST.

Mission

Open Data Science Training is a repository of freely available resources for self-directed learning in Data Science. The resources include lessons, curated datasets and exercises with a focus on practical and immediate applicability.

The goal is to enable sufficiently motivated and technically capable learners to use data science techniques in their applications without having to go get a degree in data science.

The target audience is anyone who is motivated to learn data science and is not afraid of a small amount of math, which is introduced gently.

The expectation is that the user is open to expanding and enhancing their math skills as a result of using these materials. Supplementary math learning resources are either provided or pointed too.

The approach is breadth first, so that a user may learn at a level comfortable to them or can dig deeper, but useful results and techniques are available with a small amount of learning, and not postponed for a semester or after a degree program.

The long term goal is to evolve towards a self-sustaining self-organizing community around Data Science learning, with ongoing knowledge creation via forums and community driven curation.

The Content

The current state of the content is beta. It is based on IPython Notebook and Python based Data Science libraries.

Background

If you are unfamiliar with IPython Notebook you can start with http://ipython.org/notebook

Prerequisites

One of the following distributions is needed.

  • EPD Free Enthought Python Distribution from http://enthought.com
  • Anaconda Python from http://continuum.io

Development has been done on v 1.5 of Anaconda distribution but EPD Free should work just as well.

Please note that even if you have Python installed it is important to have one of these distributions installed and the binary for this installation in your path. This is because these distributions come packaged with all the supplementary libraries needed and these have been historically difficult to install separately.

Installation

These steps assume you have installed one of the distributions mentioned in prerequisites.

From the zip for tar file
  • download and unpack the file to a directory called opendst
  • cd to the 'notebooks' subdirectory
  • start IPython Notebook 'ipython notebook --pylab=inline'
From the git repo
  • clone the repo
  • cd to 'notebooks'
  • start IPython Notebook 'ipython notebook --pylab=inline'
